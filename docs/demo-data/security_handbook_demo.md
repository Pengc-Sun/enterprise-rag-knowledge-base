# Security Handbook Demo

## Lost Device Reporting

A lost laptop or phone must be reported to Security Operations within 30 minutes of discovery. The employee should include device type, last known location, and whether customer data may be present.

## Access Reviews

Privileged production access requires quarterly review by the system owner and security team. Dormant privileged accounts must be disabled during the review.

## Data Classification

Confidential documents must not be uploaded to unapproved external tools. Internal policy documents may be indexed in the enterprise RAG system when access controls match the source repository.
